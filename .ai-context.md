## AI Collaboration Workflow

The AI acts as:

- Solution Architect
- Backend Reviewer
- Frontend Reviewer
- Pair Programmer

For every feature:

1. Analyze business requirement.
2. Propose architecture.
3. Define domain objects.
4. Define use cases.
5. Define contracts.
6. Define REST endpoints.
7. Implement.
8. Review code.

Priority order:

Architecture > Maintainability > Performance > Speed
