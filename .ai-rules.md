# Wrappedfy Specific Rules

## Frontend Stack

Official frontend stack:

- React
- TypeScript
- Vite
- Tailwind
- React Router
- Motion
- Recharts

Do not propose Angular migration unless explicitly requested.

The Figma Make project is already the foundation of the frontend.

## Backend Stack

Official backend stack:

- Java 21
- Spring Boot 3+
- PostgreSQL
- Maven

## Architecture Review Rules

Always review features using:

DDD Pragmático + Hexagonal Leve

Layers:

api
application
domain
infrastructure

Rules:

- Controller cannot contain business rules.
- UseCase contains orchestration.
- Domain contains business concepts.
- Infrastructure contains integrations.

## Feature Development Flow

For every new feature:

1. Business analysis
2. Domain modeling
3. Contract definition
4. Endpoint definition
5. Use Case implementation
6. Tests
7. Review

## Spotify Integration Rules

Spotify is the primary external integration.

Main responsibilities:

- OAuth Authentication
- User Profile
- Top Artists
- Top Tracks
- Top Genres
- Wrapped Generation

## Code Review Checklist

Always validate:

- SOLID
- Single Responsibility
- Low Coupling
- High Cohesion
- Naming Consistency
- Testability
- Dependency Direction
